# Proyecto de Generador de Horarios

Este proyecto será desarrollado en Python.
Organizador Automático de Actividades

Permite ingresar actividades, personal disponible, recursos, y tiempos; luego valida la información y genera un horario final.

El programa también permite exportar el horario en formato TXT y PDF. 

FUNCIONES

Ingreso de datos:
- Actividades
- Personal disponible
- Recursos disponibles
- Tiempos disponibles

Validaciones:
- Verificación de disponibilidad del personal
- Verificación de recursos por actividad

Asignación de horario:
- Cruza las validaciones y genera una propuesta de horario

Visualización:
- Muestra el horario en pantalla
- Exporta el horario a texto o PDF
